# Ce que PROSPECTRE sait faire

PROSPECTRE lit des packs documentaires et les rend explorables comme graphes. Cela paraît modeste. La modestie est une bonne ruse quand elle fonctionne.

| Fonction | Usage |
| --- | --- |
| Graphe typé | Voir les objets, leurs familles et leurs relations. |
| Fiches Markdown | Lire un contenu riche sans perdre les métadonnées. |
| Schéma embarqué | Déclarer types, champs, couleurs, tailles et références. |
| Filtres | Tester une lecture par familles d’objets. |
| Édition locale | Corriger avant export, sans publier trop vite. |
| Import/export | Transporter un corpus complet comme dossier de travail. |
| Coprésence | Discuter autour du corpus sans confondre échange et contenu final. |
| Diagnostic graphe | Observer volume, liens, types, couverture et hubs. |

Ce n’est pas un gestionnaire universel de vérité. C’est un appareil de lecture. Les appareils de lecture sérieux changent parfois ce que l’on croyait avoir écrit.
