# Modèle, visibilité et granularité

Le modèle d’un pack ne doit pas seulement décrire le corpus. Il doit organiser une première lecture.

## Trois statuts

| Statut | Usage |
| --- | --- |
| Nœud visible | Objet que l’on veut explorer dans le graphe |
| Fiche non centrale | Objet utile en détail, mais discret dans la première vue |
| Contenu parent | Information qui n’a pas besoin d’identité propre |

La granularité parfaite n’existe pas. La granularité défendable, oui : elle explique pourquoi chaque nœud visible est là.
