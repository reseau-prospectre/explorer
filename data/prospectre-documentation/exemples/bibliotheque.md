# Bibliothèque

Plusieurs packs peuvent composer une collection de cas d’usage.

Un exemple sert à décider plus vite. Il ne remplace pas le jugement, mais il évite de répéter les mêmes erreurs avec un air neuf.

| Point | Réponse |
| --- | --- |
| Cas | Plusieurs packs peuvent composer une collection de cas d’usage. |
| À regarder | Le titre, la granularité, le lien et l’effet dans la première vue. |
| Décision | Garder, fusionner, renommer, masquer ou déplacer dans une fiche parent. |

## Lecture rapide

1. Comparer le cas à votre pack.
2. Identifier la décision transférable.
3. Refuser la copie si le contexte ne suit pas.

## Critère

Un exemple n’est pas une décoration. Il réduit le temps nécessaire pour prendre une décision correcte.
