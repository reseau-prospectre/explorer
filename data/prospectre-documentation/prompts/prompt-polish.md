# Prompt polish — rendre un pack lisible

```text
Audite ce pack PROSPECTRE comme un lecteur exigeant.

Cherche :
- labels trop techniques ;
- codes visibles qui devraient rester internes ;
- types trop nombreux ;
- nœuds visibles sans rôle de lecture ;
- relations décoratives ;
- fiches qui mélangent contenu final et commentaire de conception ;
- tables ou listes qui pourraient mieux structurer le contenu.

Propose ensuite :
1. les types à masquer ou fusionner ;
2. les labels à réécrire ;
3. les relations à supprimer ou renommer ;
4. les contenus à déplacer dans des fiches parentes ;
5. une cible de nombre de nœuds visibles.
```
