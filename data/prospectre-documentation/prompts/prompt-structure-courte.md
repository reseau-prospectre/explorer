# Prompt court — trouver la structure

```text
À partir du corpus fourni, propose une structure de pack PROSPECTRE.

Donne :
- 4 à 8 types maximum ;
- le rôle de chaque type ;
- les types visibles dans le graphe ;
- les types à garder comme contenu ou preuve ;
- 20 à 80 nœuds visibles attendus ;
- 5 exemples de titres humains ;
- 5 relations réellement utiles.

Ne rédige pas encore le pack. Je veux d’abord valider l’architecture.
```
