# Prompt audit — vérifier un pack

```text
Vérifie ce pack PROSPECTRE avant diffusion.

Contrôle :
- cohérence des types ;
- titres lisibles ;
- liens cassés ou trop nombreux ;
- nombre de nœuds visibles ;
- présence de contenu obsolète ou métacommentaire ;
- exemples réellement utiles ;
- première vue du graphe.

Classe les problèmes par priorité et propose des corrections concrètes.
```
