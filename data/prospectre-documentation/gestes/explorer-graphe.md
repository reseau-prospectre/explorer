# Le jeu de l’exploration

Le graphe est un jeu d’imitation assez simple : il imite la structure de votre corpus, mais il ne fait pas semblant d’être plus intelligent que lui.

## Règles du jeu

1. Les gros nœuds indiquent les objets directeurs.
2. Les couleurs indiquent les types déclarés par le pack.
3. Les liens indiquent des références, pas des affinités décoratives.
4. Le filtre sert à poser une question au corpus : « que reste-t-il si je ne regarde que ceci ? »

## Bonne pratique

Commencez large, puis retirez. Un graphe se lit souvent par soustraction : cacher les détails, garder les pivots, rouvrir les fiches fines quand elles deviennent nécessaires.

## Mauvais signe

Si tout doit être affiché pour que le pack soit compréhensible, le modèle est probablement trop plat. Une fiche parent doit parfois porter le détail au lieu de créer vingt nœuds polis mais muets.
