# Premiers pas

Cette procédure couvre la première session, depuis l’ouverture de l’application
jusqu’à l’export du travail.

## 1. Comprendre les zones de l’écran

| Zone | Fonction |
|---|---|
| Barre supérieure | recherche, filtres, coprésence, transferts et profil |
| Graphe central | vue spatiale des fiches et relations |
| Barre verticale | réinitialisation, zoom et recadrage |
| Panneau droit | détails, édition et échanges de la fiche sélectionnée |
| Tiroir inférieur | contexte, indicateurs et tableau des éléments |

## 2. Charger un projet

Deux méthodes sont possibles :

1. ouvrir **Importer / exporter**, puis **Importer un projet** ;
2. glisser une archive ZIP ou des fichiers sur la fenêtre.

Le ZIP est recommandé : il conserve le manifeste, les sous-dossiers, le schéma
et les images locales.

## 3. Lire le graphe

Chaque couleur correspond à un type défini dans le modèle. La taille des nœuds
peut dépendre du type et de certaines métriques. Les lignes représentent les
relations déclarées ou déduites des métadonnées.

Cliquez sur une pastille de filtre pour masquer ou réafficher un type. La
recherche porte sur les libellés et le contenu indexé.

## 4. Ouvrir une fiche

Sélectionner un nœud ouvre le panneau droit :

- l’onglet **Détails** présente le résumé, le Markdown et les relations ;
- l’onglet **Échanges** présente les commentaires lorsque la collaboration est
  activée ;
- le commutateur **Modifier** ouvre l’éditeur local.

## 5. Sauvegarder

La session est conservée dans le navigateur, mais l’archive ZIP reste la
sauvegarde portable de référence.

> Exportez régulièrement le projet après une séance importante. La sauvegarde
> locale dépend du navigateur et de l’appareil.

## Checklist de première session

- [ ] Le projet apparaît dans le tiroir inférieur.
- [ ] Les filtres correspondent aux types attendus.
- [ ] Une fiche s’ouvre correctement.
- [ ] Les relations principales sont visibles.
- [ ] Un export ZIP a été testé.

La fiche suivante, **Explorer le graphe**, détaille les gestes de navigation.
