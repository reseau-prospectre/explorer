# Fabriquer un pack

La tentation normale est de tout extraire. Elle est compréhensible et presque toujours mauvaise.

## Méthode

1. Identifier les objets que l’on veut voir dans le graphe.
2. Identifier les informations qui doivent rester dans les fiches.
3. Donner à chaque type une couleur, une taille et un rôle.
4. Créer peu de relations, mais les créer sérieusement.
5. Tester la première ouverture : si le graphe ressemble à un sac de nœuds, revenir au modèle.

## Ratio raisonnable

| Corpus | Nœuds visibles conseillés |
| --- | --- |
| Document court | 15 à 40 |
| Projet de travail | 40 à 90 |
| Référentiel pédagogique | 60 à 140, avec détails masqués |
| Corpus encyclopédique | plus, seulement si les filtres racontent quelque chose |
