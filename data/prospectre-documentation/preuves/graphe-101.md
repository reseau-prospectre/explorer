Ce pack vise 101 fiches. Le nombre est arbitraire, donc utile : il empêche la documentation de se réfugier dans le minimalisme.

La question n’est pas : « 101 est-il beaucoup ? » La question est : « le graphe reste-t-il navigable quand il cesse d’être un jouet ? » Ici, la structure doit répondre oui, preuves à l’appui.
