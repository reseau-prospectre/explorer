# Documentation autoportée

La documentation est elle-même un pack et prouve le format par usage.

Une preuve n’est pas une proclamation. C’est une manipulation, un format, une trace, un résultat que l’on peut vérifier dans le pack.

| Point | Réponse |
| --- | --- |
| Principe testé | La documentation est elle-même un pack et prouve le format par usage. |
| Observation | Un pack manipulable vaut mieux qu’un schéma héroïque. |
| Critère | La preuve doit survivre à l’export et à la relecture par un tiers. |

## Manipulation

1. Ouvrir la fiche liée.
2. Tester le chemin dans le graphe.
3. Vérifier que la preuve reste lisible hors conversation.

## Critère

Une preuve acceptable n’a pas besoin d’être crue. Elle accepte d’être inspectée.
