# Formats ouverts

La force de cette collection tient à sa sobriété.

| Format | Rôle |
| --- | --- |
| CSV | Référentiel de compétences transportable. |
| JSON | Index ou catalogue de fichiers. |
| Markdown | Fiche lisible dans et hors PROSPECTRE. |
| GitHub Pages | Hébergement public stable. |
| Manifest PROSPECTRE | Modèle explicite et graphe navigable. |

La chaîne n'est pas magique. Elle est mieux que magique : elle est inspectable.
