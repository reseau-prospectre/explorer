# Double lecture eldoom / PROSPECTRE

La complémentarité est nette.

| Outil | Rôle |
| --- | --- |
| GitHub Pages | Héberger le CSV public et stable. |
| eldoom CBE | Prévisualiser le référentiel Moodle CSV dans son format naturel. |
| PROSPECTRE | Convertir, cartographier, commenter, comparer et gouverner le référentiel. |

Le bon parcours n'est pas de choisir un camp. Il est de passer de l'un à l'autre sans casser la source.
