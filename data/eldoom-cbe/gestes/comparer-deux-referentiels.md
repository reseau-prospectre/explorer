# Comparer deux référentiels

La collection contient des doublons, variantes, traductions, versions locales et référentiels parents. Ce n'est pas un défaut. C'est exactement ce qu'une bibliothèque doit rendre visible.

Comparer deux CSV dans PROSPECTRE revient à regarder :

- le nombre de niveaux ;
- la profondeur hiérarchique ;
- les compétences transversales ;
- les titres humains ou cryptiques ;
- les champs vides ;
- les dépendances implicites.

Le tableur montre les lignes. Le graphe montre les choix.
