# Ouvrir un CSV

Chaque fiche référentiel contient trois liens utiles : le CSV direct, la prévisualisation eldoom CBE et la source GitHub.

1. Ouvrir d'abord eldoom pour reconnaître le référentiel.
2. Ouvrir le CSV direct si l'on veut importer ou vérifier le fichier brut.
3. Revenir à GitHub si l'on veut contrôler l'origine, l'historique ou le dépôt.

La fiche n'est pas un duplicata. C'est un aiguillage.
