# eldoom CBE - bibliothèque de référentiels pour PROSPECTRE

Ce pack est un hommage outillé : il transforme la galaxie publique eldoomCBE en bibliothèque navigable pour PROSPECTRE.

Il ne recopie pas les référentiels. Il les catalogue, les relie et prépare leur exploration : GitHub conserve les fichiers, eldoom les prévisualise, PROSPECTRE les convertit en cartes de compétences.

## Contenu

- 158 référentiels CSV publics.
- 10 dépôts sources contenant des CSV.
- 4 catalogues JSON repérés.
- Des portes d'entrée, gestes, coulisses et preuves pour lire cette collection sans se perdre dans l'abondance.

## Parcours recommandé

1. Ouvrir [La bibliothèque qui manquait](entity:porte:la-bibliotheque-qui-manquait).
2. Explorer [Programmes nationaux de BUT](entity:collection:mcf-but).
3. Comparer avec [Archive MoodleNet](entity:collection:moodlenet-repo).
4. Tester un CSV dans eldoom CBE, puis l'importer dans PROSPECTRE.
5. Revenir au graphe : le catalogue doit devenir une carte, pas une étagère.
